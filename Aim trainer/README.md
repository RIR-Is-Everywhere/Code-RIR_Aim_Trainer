# Cyberpunk Aim Trainer

A simple, browser-based Aim Trainer to practice cursor precision and reaction time. Built with HTML, CSS, and Vanilla JavaScript, featuring a sleek, responsive Cyberpunk aesthetic grid.

## Features

- **Cyberpunk UI**: Glowing neon buttons, glitch effects, and CRT grid backgrounds.
- **Multiple Difficulties**: Easy, Medium, and Hardcore modes adjusting target size, spawn speed, and shrink rate.
- **Dynamic Scoring System**: Points are awarded depending on how quickly a target is clicked and streak combos.
- **Local Highscore Storage**: Automatically saves the highest score locally to be retrieved upon subsequent sessions.
- **Sound Effects**: Firing and hitting targets integrate engaging sound effects.

## How to Play

1. **Open the Game**: Simply open `index.html` in your favorite web browser.
2. **Select Difficulty**: Choose between Easy, Mid-Tier, and Hardcore from the main menu.
3. **Initialize**: Click "INITIALIZE" to start the 30-second countdown.
4. **Hit Targets**: Click the shrinking circles before they disappear. Missing or clicking the background breaks your streak.

## Deployment to GitHub Pages

This project is completely static, meaning it can be hosted for free on GitHub Pages.

1. Create a new public repository on your GitHub account.
2. Upload all the project files (including the `Image` folder, `index.html`, `style.css`, `script.js`, and this `README.md`) to the repository.
3. Navigate to **Settings > Pages** on your GitHub repository.
4. Under "Build and deployment", set the source to **Deploy from a branch**.
5. Select the `main` or `master` branch and the `/root` folder, then click **Save**.
6. Wait 1-2 minutes. Your Aim Trainer should now be live on the provided GitHub Pages URL!
